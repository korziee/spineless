Running projects:

spineless-server
----------------
```
cd spineless-server
yarn dev
```
Server will start at http://localhost:3000/

spineless-website
----------------
Ensure jekyll & bundler are installed
```
gem install jekyll bundler
```
Install dependencies
```
bundle install
```

Start server
```
bundle exec jekyll serve
```
Server will start at http://localhost:4000/
